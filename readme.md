# anjum-wp-task

This is a plugin for WP task.

## Installation

```bash
npm install
```

## Watch

```bash
npm run watch
```

## Production Build

```bash
npm run build
```

## Development

```bash
npm run dev
```

## Dependencies
- [Vue.js](https://vuejs.org/)
- [vue-router](https://router.vuejs.org/)
- [@wordpress/hooks](https://www.npmjs.com/package/@wordpress/hooks)
